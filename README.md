# Web App ASP.NET Core Project

Bu proje, ASP.NET Core kullanılarak geliştirilmiş bir web uygulamasıdır. 

## Hakkında
Bu web uygulaması, Code-First yaklaşımı ile geliştirilmiş olup veri tabanı yönetimi ve web geliştirme süreçlerini kapsamaktadır.

## Kullanılan Teknolojiler
- **Framework:** ASP.NET Core
- **Database:** SQL Server
- **Architecture:** Code-First
- **Language:** C#

## Özellikler
- Dinamik veri tabanı yönetimi.
- ASP.NET Core ile modern web arayüzü.
